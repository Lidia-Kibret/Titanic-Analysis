# 📊 Titanic Data Analysis & Custom Dataset Project

## 🚀 Overview
This project demonstrates fundamental **data analysis skills using Python and Pandas**.  
It is divided into two parts:
1. Creating a custom dataset  
2. Analyzing the Titanic dataset  

---

## 🔹 Part 1: Custom Dataset
In this section:
- A dataset is created using a Python dictionary  
- Converted into a Pandas DataFrame  
- Custom index values are applied  
- The dataset is displayed and explored  

---

## 🔹 Part 2: Titanic Dataset Analysis
In this section:
- The Titanic dataset is loaded from a CSV file  
- Data exploration is performed using:
  - `.head()`
  - `.info()`
  - `.describe()`  
- Data cleaning is applied:
  - Missing values in **Age** are filled with the median  
  - Missing values in **Embarked** are filled with the mode  
  - The **Cabin** column is removed  
  - Duplicate rows are dropped  

---

## 📈 Analysis Performed
- Survival rate by **Gender**  
- Survival rate by **Passenger Class**  
- Average age per class  
- Survival rate by **Age Groups**  
- Filtering specific groups:
  - Female survivors  
  - Child survivors  
  - First-class survivors  

---

## 🔍 Key Insights
- Females had a higher survival rate than males  
- First-class passengers were more likely to survive  
- Children had a higher chance of survival  
- The highest survival group was **female passengers in first class**
